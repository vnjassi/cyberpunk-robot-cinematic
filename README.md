# Cyberpunk Robot Cinematic (Blender Python)

Professional 3D cyberpunk cinematic generator using Blender Python API.

## Features
- Cyberpunk street
- Neon lights
- Cinematic tracking camera
- 4K render
- Animation ready

## Run

Open Blender → Scripting → Run main.py
